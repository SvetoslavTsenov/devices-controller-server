# Server to control simulatos, emulators and real devices.

## Install

Install the node packages via:

`$ npm install`

Compile server 

`$ npm run compile`

Install mogodb

`$ brew update`
`$ brew install mongodb`

Create database default folder

`mkdir -p data/db`

Run mongodb

`mogod`

## Starting

To start the server run:

`$ npm start`
